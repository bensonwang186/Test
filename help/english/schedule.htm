PowerPanel® Personal can be configured to automatically start up and shut down a connected computer and equipment. The setting is for a weekly schedule. A shut down must be scheduled prior to a restart.

  • Column "ON" shows the UPS turn on time.
  • Column "OFF" shows the UPS turn off time.
  • You can choose the UPS turn on time in the "ON" column.
  • You can choose the UPS turn off time in the "OFF" column.
  • After selecting the time, you must check on the (Act.) option to activate the setting.
  • Once you have selected the (N.R) option, the UPS will not auto restart even if you have set a restart time.
Note: (N.R) stands for Never Restart.
Note: You have to manually restart the UPS if the (N.R) button is selected.
Note: When you select a start up time you must also select a shut down time.

The space under the Schedule column is a description area that will show the schedule settings.

Note : This is not available for all UPS models.
