The system tray icon displays the status of the UPS and provides a quick way to start the main user interface.
To display the current battery status place the cursor over the system tray icon.

  • The system tray icon displays the status of the UPS:

    		The UPS is working normally.
    		The UPS is in battery mode.
    		PowerPanel® Personal is unable to connect your UPS due to a loss of communication.

    When you move the mouse cursor over the icon, you can view the tip text. Tips show you whether the UPS is charging, operating on battery power, communicating properly, or if it's fully charged.

    The available tips are as follows:
    • PowerPanel Personal

      The UPS is in a normal state.

    • PowerPanel Personal Service is not running

      PowerPanel's service is not running.

    • Computer unable to communicate with UPS

      The PowerPanel® service can not establish communication with the UPS. Check the communication data cable for connectivity.

    • UPS charging - PowerPanel Personal

      The UPS is charging when battery capacity is less than 100%.

    • UPS is on battery power - runtime remains less than x minute

      The UPS has encountered a power problem and is using batteries to supply power The remaining runtime is also indicated.

  • To launch the main user interface, double-click or right-click the icon and select Open Application.
