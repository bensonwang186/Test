The main screen of PowerPanel® Personal shows the following:

  1. Power source
    This icon shows whether the current power source is AC utility power or battery power.
    		The current power source is AC utility power.
    		The current power source is battery power.
    		PowerPanel® Personal can't detect the current power source due to a loss of communication*.

  2. Battery Capacity*
    This icon shows the remaining battery capacity.
    		The battery is charging and shows the current charge as a percentage of its capacity.
    		The UPS is using its battery to supply power and is discharging.

  3. Estimated Runtime (or Time To Shutdown)*:
    The remaining runtime the UPS can supply (The time remaining until shutdown).
    		The runtime of the UPS* when the power source is AC utility power.
    		The runtime of the UPS, or the time until hibernation or shutdown begins, when the power source is battery.

  4. Features column
    • Monitor : Click for the Monitor screen.
    • Configuration : Click for the Configuration screen.
    • Help : Click for the Help screen.

  5. Functions bar

    This bar displays different functions available for the selected feature.

  6. Workspace

    This area displays information specific to the selected feature.

  7. Status Bar The UPS is working normally : The UPS is ready to supply power if a power problem occurs. PowerPanel® Personal Service is not ready : This message will appear when the PowerPanel® Personal service is not running. Unable to establish communication with UPS : PowerPanel® Personal is unable to monitor your UPS because communication has been lost.

*Note : This is not available for all UPS models.
