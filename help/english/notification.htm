When a power event occurs, PowerPanel® Personal provides two ways to warn users:

Software Sounds

If Enable PowerPanel® notification sounds is selected PowerPanel® Personal will generate audible sounds to warn users when a power event occurs.

UPS Alarms*The UPS will provide an audible alarm.

  • Enable alarms at all times
    When a power event occurs the UPS will beep.

  • Disable alarms at all times
    When a power event occurs the UPS will not beep.

Email Notification

  • Service provider is Gmail
    Select Gmail option in the Service Provider field. Click Request to ask Google to grant the Gmail account authorization to send a mail notification. Enter the sender name and click Apply to save the settings.

  • Service provider is Other
    Select Other option in the Service Provider field and enter the information of the E-mail account.

  • Activate
    Specify whether the Agent/Client can use E-mail to send a notification to recipients.

  • Service Provider
    Select the service provider of E-mail account.

  • SMTP server address
    Configure the SMTP server used to send E-mail to a recipient.

  • Secure connection
    Set secure connection for the SMTP service to send the E-mail.

  • Service port
    Set the port number for the SMTP service.

  • Sender name & Sender E-mail address
    Enter the sender information for the E-mail.

  • Authentication
    Configure whether the SMTP server requires authentication or not. If authentication is required, enter the necessary account and password.

  • Account
    Set the account to access the SMTP server.

  • Password
    Set the password for the account.

*Note : This is not available for all UPS models.
