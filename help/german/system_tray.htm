Das Symbol in der Systemzeile zeigt den Status der USV und verfügt über eine Schnellstartfunktion der Benutzeroberfläche.
Fahren Sie mit den Mauszeiger über das Symbol, wird der aktuelle Batteriestatus angezeigt.

  • Das Systemzeilen-Symbol zeigt Ihnen den Status der USV:

    		Die USV arbeitet normal.
    		Die USV arbeitet im Batterie-Modus.
    		Ihr Rechner kann keine Kommunikation zur USV aufbauen.

    Wenn Sie den Mauszeiger über das Symbol fahren, erscheint eine Quick-Info. Diese Informationen zeigen, ob Ihre USV aufgeladen wird, im Batterie-Modus arbeitet, korrekt kommuniziert oder voll geladen ist.

    Folgende Informationen stehen zur Verfügung :
    • PowerPanel Personal

      Die USV arbeitet normal.

    • PowerPanel Personal Service arbeitet nicht

      Der Hintergrunddienst von PowerPanel arbeitet nicht.

    • Rechner kann nicht mit USV kommunizieren

      PowerPanel kann keine Kommunikation zur USV aufbauen, bitte prüfen Sie das Serielle- oder USB-Kabel.

    • USV lädt - PowerPanel Personal

      USV wird geladen, dabei befindet sich die Batterie nicht voll im Online-Modus.

    • USV arbeitet im Batterie-Modus - Laufzeit weniger als x Minuten

      Die USV reagiert auf ein Störung und versorgt das System über die Batteriespannung. Die Restlaufzeit wird angezeigt.

  • Wollen Sie die Benutzeroberfläche aufrufen, klicken Sie zweimal auf das Symbol oder klicken Sie rechts und wählen Sie Anwendung öffnen.
