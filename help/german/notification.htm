Wenn ein Problem mit der Stromversorgung auftritt, bietet PowerPanel Personal zwei Wege der Benachrichtigung :

Software-Warnsignal

Wenn unter PowerPanel ein Warnton ausgewählt wurde, warnt die PowerPanel Personal den Benutzer mit einem Signalton, wenn eine Versorgungsstörung auftritt.

USV-Alarm*Die USV gibt einen Signalton aus. Dieser unterscheidet sich von dem Software-Signalton.

  • Alarmton immer aktiv.
    Tritt ein Versorgungsfehler auf, sendet die USV ein Alarmsignal.

  • Alarmton nie aktiv.
    Tritt ein Versorgungsfehler auf, ertönt kein Alarmsignal.

Email Benachrichtigung

  • Dienstanbieter ist Gmail
    Wählen Sie im Feld Dienstanbieter die Option Gmail aus. Klicken Sie auf Autorisieren, um Google zu bitten, die Gmail-Kontoberechtigung zu erteilen, um eine E-Mail-Benachrichtigung zu senden. Geben Sie den Namen des Absenders ein und klicken Sie auf Übernehmen, um die Einstellungen zu speichern.

  • Dienstleister ist Sonstiger
    Wählen Sie im Feld Dienstanbieter die Option Andere Option aus und geben Sie die Informationen des E-Mail-Kontos ein.

  • Aktivieren
    Geben Sie an, ob der Agent / Client E-Mail verwenden kann, um eine Benachrichtigung an den Empfänger zu senden.

  • Dienstanbieter
    Wählen Sie den Dienstanbieter des E-Mail-Kontos aus.

  • SMTP-Serveradresse
    Konfigurieren Sie den SMTP-Server, der zum Senden von E-Mails an einen Empfänger verwendet wird.

  • Sichere Verbindung
    Sichere Verbindung für den SMTP-Dienst festlegen, um eine E-Mail zu senden.

  • Service-Port
    Stellen Sie die Portnummer für den SMTP-Dienst ein.

  • Absender Name & Absender E-Mail-Adresse
    Geben Sie die Absenderinformationen für die E-Mail ein.

  • Authentifizierung
    Konfigurieren Sie, ob der SMTP-Server eine Authentifizierung erfordert oder nicht. Wenn die Authentifizierung erforderlich ist, geben Sie das erforderliche Konto und das Passwort ein.

  • Konto
    Stellen Sie das Konto für den Zugriff auf den SMTP-Server ein.

  • Kennwort
    Stellen Sie das Kennwort für das Konto ein.

*Hinweis : Diese Funktion ist nicht für alle USV Systeme verfügbar.
